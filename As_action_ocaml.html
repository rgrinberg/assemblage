Module As_action_ocaml

module As_action_ocaml: sig .. end
Built-in actions for OCaml.

The strategy is the following. At this level we only deal with unlifted actions and we are concerned about the command options that deal with inputs and output specification (e.g. -bin-annot or packages, but not -g).
